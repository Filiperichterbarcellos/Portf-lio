# Portfolio

O projeto consiste no desenvolvimento de uma plataforma web inspirada no Exevo Pan, um sistema amplamente utilizado na comunidade do jogo Tibia para análise de mercado, gerenciamento de personagens e outras funcionalidades voltadas para jogadores. O objetivo é criar uma solução interativa e moderna que forneça recursos avançados para os usuários, como estatísticas em tempo real, histórico de preços e busca otimizada.

Para o desenvolvimento, será utilizada a linguagem Java no back-end, garantindo robustez e escalabilidade. No front-end, pretende-se adotar uma tecnologia moderna como React.js ou Vue.js, proporcionando uma interface dinâmica e responsiva. Além disso, será explorada a integração com APIs externas para obtenção e manipulação de dados relevantes.

Este projeto tem como foco a aplicação prática dos conceitos aprendidos na disciplina, abordando aspectos como arquitetura de software, consumo de APIs, design de interface e boas práticas de desenvolvimento.
